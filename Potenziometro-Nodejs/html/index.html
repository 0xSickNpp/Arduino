PROGETTO DI TELECOMUNICAZIONI



INTRODUZIONE


Il progetto scelto, come da titolo, sarà un circuito che accenderà un determinato led in base al valore letto tramite un potenziometro.

PROCEDIMENTO


Per prima cosa abbiamo assemblato il circuito sull' ARDUINO UNO. Poi abbiamo scritto del codice in C dove in base al valore del potenziometro, l' Arduino UNO invia al dispositivo seriale (RASPBERRY PI) un valore X. Il codice scritto in PYTHON, che si interfaccia all'Arduino UNO come dispositivo seriale, registra in una variabile il valore ricevuto e invia una richiesta web tramite metodo POST ad una pagina web su un server in cui è attivo uno script scritto in JAVASCRIPT usando il framework NODEJS (con il modulo EXPRESS) che successivamente registrerà il valore insieme all'orario dell'arrivo della richiesta in una variabile. I risultati vengono mostrati in tempo reale in un' altra pagina che periodicamente invierà una richiesta web alle API del sito per ottenere gli ultimi valori. Tutto ciò avviene su un raspberry pi che lavora come SERVER.
